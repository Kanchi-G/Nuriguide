Nuriguide System

📌 Project Overview

Nuriguide System is a web-based health and nutrition recommendation platform that suggests personalized meals and diet plans based on user health parameters such as age, weight, BMI, and health conditions. The system helps users maintain a healthy lifestyle through smart meal recommendations.

🚀 Features
User-friendly web interface
Personalized meal recommendations
Health parameter analysis
BMI-based diet suggestions
Secure login and authentication system
Downloadable diet plan PDF
ML-based prediction for health recommendations
Responsive and interactive dashboard
🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: Python / Flask
Database: MySQL / SQLite
Machine Learning: Scikit-learn
Tools & Libraries: Pandas, NumPy
📂 Project Workflow
User enters health details
System analyzes the input data
ML model predicts suitable meal recommendations
Personalized diet plan is generated
User can download the diet plan in PDF format
🎯 Objective

The main objective of the Nuriguide System is to provide users with healthy and personalized nutrition guidance using technology and machine learning.

📈 Future Enhancements

Mobile application integration
